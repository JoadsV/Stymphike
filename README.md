# Stymphike Hunter

A RuneLite plugin for hunting [Stymphikes](https://oldschool.runescape.wiki/w/Stymphike) in Vampyrium.

## Features

- **Catch notifications** — a chat message (and optional system notification) each time you successfully catch a stymphike.
- **Flee alerts** — tells you when a stymphike flees with your bait (i.e. you dealt under 10 damage), which produces no game message on its own.
- **Baited-tree markers** — a circle marker on every tree you have baited, read live from the game's own tree state, so it appears exactly when the bait lands and clears when it is consumed (caught, fled, or eaten while you were away).
- **Stymphike outline** — outlines the bird the moment it appears so you can spot and spear it quickly.
- **Hiding status** — an on-screen indicator showing whether you are hidden in a bush (stymphikes only approach while you are hidden). It only reminds you to hide when you have a baited tree waiting, and hides entirely outside the hunting area.

## Configuration

- Notify on catch / flee / when exposed
- Highlight baited trees and/or all trees, and outline the stymphike, with configurable colours
- Show/hide the hiding-status box

## Requirements

The same as hunting stymphikes normally: 82 Hunter, completion of *The Blood Moon Rises*, and makeshift or hunter's spears.
