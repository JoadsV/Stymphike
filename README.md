# Stymphike Hunter

A RuneLite plugin for hunting [Stymphikes](https://oldschool.runescape.wiki/w/Stymphike) in Vampyrium.

## Features

- **Catch notifications** — a chat message (and optional system notification) each time you successfully catch a stymphike.
- **Flee alerts** — tells you when a stymphike flees with your bait (i.e. you dealt under 10 damage), which produces no game message on its own.
- **Baited-tree markers** — marks trees you have baited with a circle so you can find them again. The mark is added once you actually reach the tree, and clears automatically when the bird is caught or flees.
- **Hiding status** — an on-screen indicator showing whether you are hidden in a bush (stymphikes only approach while you are hidden). It only reminds you to hide when you have a baited tree waiting.

## Configuration

- Notify on catch / flee / when exposed
- Highlight baited trees and/or all trees, with configurable colours and marker style (circle, tile, outline, or clickbox)
- Show/hide the hiding-status box

## Requirements

The same as hunting stymphikes normally: 82 Hunter, completion of *The Blood Moon Rises*, and makeshift or hunter's spears.
